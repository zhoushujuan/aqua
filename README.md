# aqua

###基于Node.js平台的web应用开发框架Express开发的项目，使用bootstrap

#### 步骤：
##### 下载后，进入项目文件夹执行：

```	$ npm install  
```

##### 生成node_modules文件夹后

##### 安装bower:

```$ npm install-g bower
```

##### 安装bootstrap

```bower install bootstrap
```

##### 生成bower_components文件夹后，那名字修改为bs,放到public文件夹下 

##### 启动项目：

```$ npm start
```
